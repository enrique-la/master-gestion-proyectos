# Gestión de Proyectos de Análisis de Datos — Máster en IA & Data Science

Metodologías y herramientas para gestionar proyectos de data science de extremo a extremo.

## Contenido

### Metodología CRISP-DM
- Ciclo de vida de proyectos de datos
- Calidad del dato y auditoría
- Documento CRISP-DM de ejemplo
- RapidMiner para modelado visual

### Agile & DevOps para Data Science
- Sprints y gestión de backlog en proyectos de datos
- CI/CD para modelos de ML
- Integración de MLOps en equipos ágiles

### Tecnologías Exponenciales
- IA, IoT, Blockchain y su impacto en la industria
- De series temporales a visión artificial

### Power BI
- Diseño de dashboards e informes interactivos
- Conexión a fuentes de datos
- DAX y modelado de datos
